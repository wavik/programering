 OPERATION OF 'TEMP'


This version of 'TEMP' is text instead of graphics oriented. It displays the information about the Temperature Device DS1920/DS1820 such as : TMEX Driver Version, Type Version, 1-Wire Network Address and the value of the temperature the device senses.
