This directory contains the Microsoft Visual C++ 6.00 console application RCONVC32.


OPERATION OF 'RCONVC32'


This version of RCONVC32 or 'RAY' is text instead of graphics oriented.  The program prompts for the 'L' key to be pressed.  When pressed, it provides a list of the 1-Wire network addresses on the 1-Wire network.  The addresses are given in a numbered list.  An address from that list can be chosen and that ADDRESS list of directory and files is provided in a similar list.  A file can then be selected, viewed and replaced. Note that at any time the 'L' key can be pressed again to provide the 1-Wire network address list.
