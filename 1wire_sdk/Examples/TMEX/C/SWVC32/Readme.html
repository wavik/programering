 OPERATION OF 'SWVC32'


This version of 'SWVC32' is text instead of graphics oriented. It displays the information about the DS2406 Dual Addressable Switch Plus Memory such as: TMEX Driver Version, Type Version, ROM number (1-Wire network address) and the status of the Activity Latch and the open drain outputs inside the switch. The Open drain outputs for each DS2406 on the 1-Wire network can be independently switched on or off whether there is one or many devices sharing the same 1-wire port. The numbers are given in a numbered list.  A number from that list can be chosen to update the value of the Sensed Level and to set the Flip Flop ON/OFF on switch or reset the value of the Activity Latch.
