Files of the sub-directory \Examples\TMEX\DELPHI\TIMEDL32\:

  This program is written for 32-Bit Microsoft Windows.  The source code
  is provided in Delphi 5.0.

  OPERATION OF 'TIME'

  The 'TIME' example program contains only one window. Its reads and displays
  the date/time from the DS1994's internal real time clock/calendar.

  README.TXT   - This file.

  TIMEDL32.DPR - Delphi project file.

  TIMEDL32.DOF - Delphi options.

  TIMEDL32.RES - Resource module.

  TIMEUN32.PAS - Main pascal source file for the window.

  iBTMEXPW.PAS - Pascal source file for the TMEX calls.

  TIMEUN32.DFM - Delphi form file.

  TIMEDL32.EXE - Program executable.
