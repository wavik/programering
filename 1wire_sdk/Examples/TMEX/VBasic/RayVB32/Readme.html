Files of the sub-directory \Examples\TMEX\VBASIC\RAYVB32\:

 This example of 'RAY' called 'RAYVB32' was written with Microsoft Visual
 Basic 6.00.  To load this project, start Visual Basic, select
 'File/Open Project', and select the file 'RAYVB32.MAK' found in this
 directory.  Note that this is for the 32 bit Visual Basic 6.00 compiler
 and TMEX drivers.

 Note that the operation of 'RAYVB32' is slightly different.
 'RAYVB32' does not allow more than one 1-Wire address or file to be
 selected at a time.  As soon as another 1-Wire address is selected
 then the current file list being displayed changes to reflect the
 new 1-Wire address.

 REGNUM.FRX   - Visual Basic module.
