TMEX Hardware Specific Error Return Codes Overview Group

(-1)   PORT_NOT_INITIALIZED - Specified 1-Wire network has not been initialized with a call to the TMSetup function.
(-2)   PORT_NOT_EXIST - Specified 1-Wire network nonexistent.
(-3)   NO_SUCH_FUNCTION - Function not supported.
(-12)  BCOM_FAILURE – Failure to communicate with hardware adapter
(-13)  BCOM_EVENT – An unsolicited event occurred on the 1-Wire
(-200) INVALID_SESSION - session not valid.
(-201) HS_NOT_FOUND - Hardware_Specific driver not found and is required.
