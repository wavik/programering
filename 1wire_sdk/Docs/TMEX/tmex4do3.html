TMEX Type Descriptions

General Types

short              16 bit signed integer
long               32 bit signed integer
char               8 bit signed number
unsigned char      8 bit unsigned number
far pascal         far function using pascal calling convention
void far *         generic far pointer
short far *        far pointer to a 16 bit signed integer

TMEX Specific Types

FileEntry          structure containing all of the information on a TMEX file
DirectoryPath      structure containing all of the information on a TMEX Directory path
Specification      structure containing descriptive information for each 1-Wire device based on its family code
